# YouTube FullWindow

A lightweight Chrome extension that adds a true full-window mode to YouTube. Press **W** or click the new control bar button to expand the video player until it fills your entire browser window.

YouTube's original theater mode (T) and fullscreen (F) remain untouched.

## Features

- **W key** — Toggle full-window mode
- **Control bar button** — Click the new button next to the fullscreen icon
- **Smart transitions** — Seamless switching between default, theater, full-window, and fullscreen modes
- **16 languages** — English, Chinese, Japanese, Korean, Spanish, German, French, Portuguese, Italian, Russian, Arabic, Hindi, Indonesian, Vietnamese, Thai, Turkish

## Install

Available on the [Chrome Web Store](https://chrome.google.com/webstore/detail/nndgnhmkhnldembalbpcgbecdgbgjdoo).

Or load manually:
1. Clone this repo or download the ZIP
2. Open `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked** and select the repo folder

## Permissions

Only `https://www.youtube.com/*` — required to inject the content script. No other permissions. No data collection.
