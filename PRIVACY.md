# Privacy Policy

**Last updated: July 2026**

## Data Collection

**YouTube FullWindow does not collect any personal data.**

The extension:
- Does not collect, store, or transmit any user information
- Does not use cookies or any tracking mechanisms
- Does not communicate with any external servers
- Does not inject any analytics or tracking code
- Does not access or modify any user data beyond the YouTube page DOM

## Permissions

### \https://www.youtube.com/*
This host permission is required solely to inject the content script on YouTube pages, which is the core functionality of the extension. The script only modifies the CSS and behavior of the YouTube video player to enable full-window theater mode.

## Third-Party Services

This extension does not use any third-party services, APIs, or SDKs.

## Changes to This Policy

If this policy changes in the future, the updated version will be posted here with a new revision date.

## Contact

For questions about this privacy policy, please open an issue on the project's GitHub repository.
